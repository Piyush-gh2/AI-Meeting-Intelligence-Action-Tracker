# 🎙️ AI Meeting Intelligence & Action Tracker

## Overview

AI-powered meeting intelligence platform that automatically analyzes meeting transcripts and extracts:

- Meeting summaries
- Action items
- Risks and blockers

## Features

- Transcript Analysis
- Action Item Extraction
- Risk Detection
- Streamlit Dashboard

---

## Installation

```bash
pip install -r requirements.txt
```

---

## Run

```bash
streamlit run app.py
```
